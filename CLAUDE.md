## Архитектура

**Что делаем:**
- Свой одностраничник (HTML + CSS, без сборки) — лендинг-обложка
- Бронирование пунктов — через внешний сервис **Followish** (followish.io), бесплатно, без регистрации гостей
- Сборы (лечение 30к, платье 18к) — через **Тинькофф по номеру +7 969 709 65 95**

**Почему свой сайт, а не только Followish:**
В списке есть нетоварные пункты («3 поцелуя в щёчку», тату «Люблю Улю», сборы) — шаблонный сервис показал бы их странно. Сайт даёт настроение и контекст, Followish даёт техническое бронирование.

## Структура файлов
- `index.html` — главная страница
- `styles.css` — стили
- `data.js` — массив пунктов вишлиста (редактировать тут при изменениях)
- `script.js` — рендер карточек + клики на бронирование
- `assets/` — фото, иконки
